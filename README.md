# 👨‍💻 Personal Portfolio Website

A premium, highly responsive, and visually stunning personal portfolio website built completely from scratch using **HTML5, CSS3, and Vanilla JavaScript**. Designed to showcase my skills, projects, and educational journey as a 3rd-year B.Tech Computer Science student.

## ✨ Features

- **Modern UI/UX:** Dark theme with glassmorphism effects, gradients, and floating background blobs.
- **Fully Responsive:** Flawless experience across Desktop, Tablet, and Mobile devices.
- **Custom Animations:** Smooth scroll reveals, typing text effects, animated skill progress bars, and counter animations.
- **Interactive Elements:** Custom mouse cursor, hover glow effects, and a sticky navigation bar with a scroll progress indicator.
- **Student-Focused Sections:** Tailored 'About' and 'Journey' sections highlighting academic progress, DSA problem-solving, and active tech club contributions.
- **Contact Form:** Beautifully styled contact interface.

## 🛠️ Tech Stack

- **Markup:** HTML5 (Semantic Structure)
- **Styling:** CSS3 (Custom Variables, Flexbox, Grid, Animations, Media Queries)
- **Scripting:** Vanilla JavaScript (ES6+, DOM Manipulation, Event Listeners)
- **Icons & Fonts:** FontAwesome 6, Google Fonts (Poppins & Inter)
- *No external frameworks like Bootstrap, Tailwind, or React were used.*

## 🚀 Featured Projects

1. **🎮 Modern Tic-Tac-Toe:** An interactive, responsive take on the classic game featuring a sleek UI, smooth animations, and win-state tracking logic.
2. **⛅ SkyCast Weather Dashboard:** A beautifully designed premium weather application providing real-time forecasts, atmospheric data, and dynamic location-based updates.
